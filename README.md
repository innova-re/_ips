Development of Indoor Navigation System using EcmaScript 6.

Motivation
----------

Absence of GPS


Technology adoption
-------------------

- PhoneGap

- Captures device motion in the x, y, and z direction.


Supported Platforms
-------------------

- Android
- BlackBerry WebWorks (OS 5.0 and higher)
- iPhone
